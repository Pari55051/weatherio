
# Weatherio

Weather App

## Tech Stack

**Client:** HTML, CSS, JavaScript, API (OpenWeather App API)

## Features

- Current Location
- Air Quality and Wind Speed
- 5 Day Forecast, Hourly Forecast, Current Weather
